# Homebrew Formulae

A Homebrew tap for various thoughtbot projects.

## Usage

Tap this repository:

    brew tap thoughtbot/formulae

Install the packages it contains like any other Homebrew package:

    brew install gitsh
    brew install liftoff
    brew install parity
    brew install pick
    brew install rcm
